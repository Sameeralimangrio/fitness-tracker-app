# fitness-tracker-app
Fitness Tracker App for assignment
This application is developed as part of the Mobile Application Development course. It is built using Expo CLI and React Native, with VS Code used for coding.

Displays a list of exercises

Tapping on an exercise opens a detailed view with an image and description

Users can add new exercises through a simple form

Setup Instructions:

Install Node.js and Expo CLI

Download or clone this project

Run: npm install

Start the app: npx expo start
